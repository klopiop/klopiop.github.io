0x02.02 支撑环境安装

0x02.02.a Git

项目地址：

Git下载地址：
https://git-scm.com/download/win
一个项目管理工具

  1. 选择下载

  2. 打开下载好的安装包进行安装,下一步

  3. 自己选择默认的编辑器，我是vscode

  4. 一直下一步，看到install，安装就行

0x02.02.b hugo

获取hugo：

hugo下载地址：
[https://github.com/gohugoio/hugo/releases](https://github.com/gohugoio/hugo/releases)
一个项目管理工具

  1. 选择对应包下载（选择带extended字样，包含完整拓展的）

  2. 创建hugo目录，丢进去就行（自定义）

  3. 创建环境变量，方便调用

  4. 测试安装情况
